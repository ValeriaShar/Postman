1. Создать окружение
2. Создать в нем переменные name, age, salary, weight
3. Записать в них значения, которые вы используете в запросах
4. Заменить в запросах текстом прописанные значения на вызов переменной окружения
5. Создать переменную url
6. Записать в нее IP + port (всё до эндпоинта)
7. Заменить в запросах ip и порт на переменную

1) Collections
  -> New
  -> Homework_2
  -> Add request
  -> name user_info_2
  -> POST
  -> http://162.55.220.72:5005/user_info_2
  -> Body
  -> KEY (name    age    salary    weight)
  -> save

2) Environments
  -> New
  -> Homework_2 (+ставим галочку)
  -> VARIABLE      (name   age   salary  weight)
  -> CURRENT VALUE (Mike   30    1800      80  )
  -> save

3) Collections
   -> Homework_2
   -> user_info_2  
   -> No Environment -> Homework_2
   -> Body
   -> VALUE  {{name}}   {{age}}  {{salary}}   {{weight}} 
   -> save
   -> send
 
4) Environments
  -> New
  -> url (+ставим галочку)
  -> VARIABLE (url)
  -> CURRENT VALUE (http://162.55.220.72:5005)
  -> save
 
5) Collections
  -> Homework_2
  -> Add request
  -> name  url
  -> GET
  -> No Environment -> url
  -> Enter request URL -> {{url}}/
  -> save
  -> send
  Получаем ответ:  "Hello!!"
